About me

Saya adalah mahasiswa Teknik Informatika yang berpengalaman dalam berbagai proyek pengembangan aplikasi dan sistem informasi. Saya memiliki pengalaman dalam pemrograman menggunakan Python dan Dart, dan dapat menggunakan framework Django dan Flutter untuk pengembangan layanan web dan aplikasi seluler.
